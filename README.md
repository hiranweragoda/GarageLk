# Garage.lk

**An Intelligent, Crowdsourced Marketplace for Spare Parts and Vehicle Maintenance in Sri Lanka**

Garage.lk is a comprehensive full-stack web application designed to bridge the gap between car owners, repair service providers (garages), and spare parts suppliers. By centralizing these services, the platform eliminates market inefficiencies, resolves information asymmetry, and establishes a transparent, tech-driven automotive ecosystem.

## Key Features

- **Enterprise Full-Stack Architecture:** Powered by a high-performance Java backend with Spring Boot and a relational MySQL database.
- **Role-Based Portals:** Custom interfaces and workflows for:
  - **Customers:** For locating nearby workshops, real-time appointment scheduling, and keeping vehicle digital service logs.
  - **Service Providers:** For managing booking requests, profiles, and shop inventory.
  - **System Administrators:** For managing reviews, registrations, and system health.
- **E-Commerce Spare Parts Marketplace:** Directly connects spare part vendors with mechanics and car owners.
- **Round-the-Clock Chatbot Support:** Helpful virtual assistance to guide users to appropriate mechanical services and spare parts.

## Tech Stack

- **Backend:** Java SE, Spring Boot (Spring MVC, Spring Security, Spring Data JPA/Hibernate)
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS), Bootstrap
- **Build & Dependency Tool:** Apache Maven
- **Database:** MySQL
